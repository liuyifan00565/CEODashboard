## 模块位置

经营总览页右上区域由原「本月渠道完成情况」表格升级为「渠道目标完成结构」模块。

该模块保留原位置和玻璃卡片层级，内部由「半环饼图 + 中心完成率 + 右侧渠道摘要」组成，并支持「本月 / 本年」切换。

半环饼图视觉与首页「版本情况」饼图保持同款：使用相同的银紫玫瑰/香槟低饱和色板、45%-76% 半径、49.5%/68% 中心点、圆角分段、轻阴影和前两项外侧轻标签；渠道图不再叠加独立局部光晕。

## 数据来源

- 渠道行数据继续使用前端统一方法 `getChannelCompletionRows(period)`。
- `period = month` 时读取本月完成、月目标、本月完成率和本月缺口。
- `period = year` 时读取年度累计、年度目标、年度完成率和年度缺口。
- 风险标签优先沿用渠道行 `warn` / `status` 结果；若渠道完成率低于当前整体完成基准，也标记为风险渠道。
- 当前整体完成基准 = `min(100%, 当前周期整体完成率)`。例如本月整体完成率 106.0% 时，完成率低于 100% 的渠道标记风险；本年整体完成率 41.0% 时，只标记低于 41.0% 的渠道。

## 图表口径

半环饼图表达的是「各渠道在整体目标完成中的目标内结构占比」，不是直接把完成率画成饼图。

### 总完成率低于 100%

当总完成率低于 100% 时：

- 各渠道扇区 = 当前周期渠道完成额 / 当前周期总目标。
- 灰色「未完成」扇区 = 当前周期总目标缺口 / 当前周期总目标。
- 中心数字显示真实总完成率。

### 总完成率大于或等于 100%

当总完成率大于或等于 100% 时：

- 半环主体只展示目标内结构，合计归一为 100%。
- 不额外绘制超过 100% 的扇区面积。
- 中心数字继续显示真实总完成率。
- 右侧状态展示「超额完成 x.x%」。

## 右侧摘要

右侧每个渠道展示：

- 渠道名称
- 贡献占比
- 完成率
- 风险标签

贡献占比与半环图口径保持一致：未达标时按总目标占比展示，超额时按完成结构归一展示。
